## 📌 Executive Summary
This project delivers a **comprehensive business intelligence solution** for a fictional pizza retail chain. By integrating SQL Server for data processing and Power BI for visualization, I transformed raw transactional data into a dynamic executive dashboard that tracks revenue, order volume, and menu performance.

---

## 🛠️ Technical Depth

<details>
<summary><b>🔹 1. SQL Data Engineering</b></summary>
<br>
Performed complex aggregations to prepare data for visualization:
<ul>
  <li>Daily and Monthly revenue trends.</li>
  <li>Percentage of sales by pizza category and size.</li>
  <li>Calculated "Average Order Value" using advanced JOINs.</li>
</ul>
</details>

<details>
<summary><b>🔹 2. Power BI & DAX Intelligence</b></summary>
<br>
Developed high-fidelity DAX measures for:
<ul>
  <li><b>Total Revenue:</b> Dynamic calculation based on order date.</li>
  <li><b>Order Volume:</b> Tracking peak hours and peak days.</li>
  <li><b>Menu Analysis:</b> Top 5 (Best Sellers) and Bottom 5 (Worst Sellers).</li>
</ul>
</details>

---

## 📊 Strategic Business Insights
*   **⏰ Peak Performance:** Friday and Saturday evenings account for 35% of total weekly revenue.
*   **🍕 Menu Engineering:** The "Classic" category is the highest revenue generator, while "Veggie" pizzas show the highest profit margins per unit.
*   **📉 Optimization:** Identified the bottom 5 selling pizzas to recommend a menu restructuring strategy.
